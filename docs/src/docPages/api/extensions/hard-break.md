# HardBreak
The HardBreak extensions adds support for the `<br>` HTML tag, which forces a line break.

## Options
*None*

## Commands
*None*

## Keyboard shortcuts
* Windows & Linux: `Shift` + `Enter`
* macOS: `Command` + `Enter`
