# Collaboration
Enables you to collaborate with others on one document.

## Installation
```bash
# With npm
npm install @tiptap/extension-collaboration yjs y-webrtc

# Or: With Yarn
yarn add @tiptap/extension-collaboration yjs y-webrtc
```
