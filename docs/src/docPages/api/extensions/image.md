# Image

## Installation
```bash
# with npm
npm install @tiptap/extension-image

# with Yarn
yarn add @tiptap/extension-image
```

## Settings
| Option | Type    | Default | Description              |
| ------ | ------- | ------- | ------------------------ |
| inline | boolean | false   | Renders the node inline. |
