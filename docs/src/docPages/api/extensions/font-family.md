# FontFamily

## Installation
::: warning Use with TextStyle
This extension requires the [`TextStyle`](/api/nodes/text-style) mark.
:::

```bash
# with npm
npm install @tiptap/extension-font-family

# with Yarn
yarn add @tiptap/extension-font-family
```
